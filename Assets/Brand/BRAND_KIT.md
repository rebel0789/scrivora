# Scrivora Brand Kit

Scrivora is a local voice dictation assistant for macOS. The brand should feel minimal, warm, focused, private, and precise.

## Core

- Name: Scrivora
- Tagline: Voice. Captured. Words. Elevated.
- Positioning: Private dictation for Mac. Local AI. Crafted for creators.
- Product promise: Think out loud and write with confidence without sending speech to cloud APIs.

## Logo

- Primary mark: terracotta rounded-square app icon with a white voice-flow waveform and fountain-pen nib.
- Wordmark: serif Scrivora wordmark, paired with the uppercase terracotta tagline.
- Monochrome: charcoal or white-only mark for constrained system surfaces.
- Minimum use: keep the central nib and two waveform bars recognizable at menu-bar scale.
- Menu bar mark: use the backgroundless template glyph, not the orange rounded-square app icon. macOS should render the template as dark on light menu bars and light on dark menu bars.

## Palette

| Token | Hex | Use |

| Accent / Terracotta | `#E07A4F` | Primary actions, active states, model selection |

| Muted Sage | `#6F806F` | Privacy, local, complete, and success states |

## Typography

- Display / headings: system serif in the app until Playfair Display is bundled.
- Body / UI: system sans-serif in the app until Inter is bundled.
- Use letter spacing sparingly. The tagline can use small uppercase tracking; dense settings rows should stay readable.

## UI Principles

- Keep the app local-first and privacy-forward without loud marketing copy.
- Use terracotta for the voice capture path and muted sage for privacy/local trust.
- Prefer restrained panels, thin dividers, and quiet empty states over decorative cards.
- Preserve dense macOS settings ergonomics. The brand should not slow down core dictation.
